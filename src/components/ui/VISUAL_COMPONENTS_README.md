# Whappi Visual Components Documentation\n\nThis document provides comprehensive documentation for the visual design components and placeholder elements created for the Whappi landing page.\n\n## Overview\n\nThese components were designed to create a professional, modern, and accessible landing page for Whappi with the following characteristics:\n\n- **Consistent Design System**: Unified color palette (blue, purple, pink gradients)\n- **Professional Appearance**: Polished, production-ready components\n- **Responsive Design**: Mobile-first approach with proper breakpoints\n- **Accessibility**: WCAG 2.1 AA compliant with proper ARIA labels\n- **Performance**: CSS/SVG graphics, optimized animations, minimal bundle impact\n- **Modularity**: Reusable components with variant support\n\n## Components Overview\n\n### 1. App Mockup (`app-mockup.tsx`)\n\n**Purpose**: WhatsApp-style interface showing team collaboration features\n\n**Features**:\n- Team chat interface with realistic conversation\n- Multi-platform badges (WhatsApp, Telegram, Messenger)\n- Internal notes and team assignment features\n- Professional messaging workflow visualization\n\n**Usage**:\n```tsx\nimport { AppMockup } from '@/components/ui/app-mockup';\n\n<AppMockup className=\"max-w-sm\" />\n```\n\n**Variants**: Default (detailed interface)\n\n### 2. Feature Icons (`feature-icons.tsx`)\n\n**Purpose**: Enhanced icons for features with custom styling and animations\n\n**Available Types**:\n- `team`: Team management icon\n- `integrations`: Connected systems icon\n- `security`: GDPR/privacy shield icon\n- `performance`: Performance/analytics icon\n- `analytics`: Data visualization icon\n- `communication`: Messaging icon\n\n**Special Components**:\n- `IntegrationIcon`: Multi-platform integration visualization\n- `WhatsAppIcon`, `TelegramIcon`, `MessengerIcon`: Platform-specific icons\n\n**Usage**:\n```tsx\nimport { FeatureIcon, IntegrationIcon } from '@/components/ui/feature-icons';\n\n<FeatureIcon type=\"team\" size=\"lg\" animate={true} />\n<IntegrationIcon size=\"md\" />\n```\n\n**Props**:\n- `type`: Icon type\n- `size`: 'sm' | 'md' | 'lg'\n- `animate`: Enable pulse animation\n- `className`: Additional CSS classes\n\n### 3. Trust Badges (`trust-badges.tsx`)\n\n**Purpose**: Security, compliance, and trust indicators\n\n**Available Types**:\n- `gdpr`: GDPR compliance badge\n- `madeInItaly`: Made in Italy flag badge\n- `certified`: ISO/certification badge\n- `verified`: Verification badge\n\n**Special Components**:\n- `SecurityBadge`: Dedicated GDPR badge\n- `MadeInItalyBadge`: Italian flag badge\n- `TrustIndicator`: Combined trust elements\n\n**Usage**:\n```tsx\nimport { TrustBadge, TrustIndicator } from '@/components/ui/trust-badges';\n\n<TrustBadge type=\"gdpr\" variant=\"detailed\" />\n<TrustIndicator />\n```\n\n**Variants**:\n- `default`: Full badge with icon and text\n- `minimal`: Compact version\n- `detailed`: Extended information\n\n### 4. Price Comparison (`price-comparison.tsx`)\n\n**Purpose**: Visual comparison showing Whappi vs traditional solutions\n\n**Features**:\n- Whappi (€15) vs Competitors (€300-400)\n- Savings calculation (95% cost reduction)\n- Feature comparison\n- ROI highlighting\n\n**Usage**:\n```tsx\nimport { PriceComparison, SavingsCalculator } from '@/components/ui/price-comparison';\n\n<PriceComparison variant=\"card\" />\n<SavingsCalculator />\n```\n\n**Variants**:\n- `horizontal`: Side-by-side comparison\n- `vertical`: Stacked comparison\n- `card`: Full comparison card with features\n\n### 5. Testimonial Cards (`testimonial-card.tsx`)\n\n**Purpose**: Customer testimonials and social proof\n\n**Features**:\n- Customer avatars (generated from initials)\n- Star ratings\n- Company information\n- Location details\n- Verification badges\n\n**Usage**:\n```tsx\nimport { TestimonialCard, FeaturedTestimonial } from '@/components/ui/testimonial-card';\n\n<TestimonialCard\n  name=\"Marco Rossi\"\n  role=\"CEO\"\n  company=\"TechFlow\"\n  rating={5}\n  quote=\"Amazing results!\"\n  variant=\"default\"\n/>\n```\n\n**Variants**:\n- `default`: Standard testimonial card\n- `minimal`: Compact version\n- `detailed`: Large featured testimonial\n- `placeholder`: Loading state\n\n### 6. Video Placeholder (`video-placeholder.tsx`)\n\n**Purpose**: Attractive video placeholders with demo interfaces\n\n**Features**:\n- Play button overlay\n- Duration display\n- Mock interface elements\n- Platform badges\n- Professional video player UI\n\n**Usage**:\n```tsx\nimport { VideoPlaceholder } from '@/components/ui/video-placeholder';\n\n<VideoPlaceholder\n  title=\"Whappi Demo\"\n  description=\"See the platform in action\"\n  duration=\"2:30\"\n  variant=\"detailed\"\n/>\n```\n\n**Variants**:\n- `default`: Standard video placeholder\n- `minimal`: Simple version\n- `detailed`: Full video player interface\n\n### 7. Logo Placeholders (`logo-placeholder.tsx`)\n\n**Purpose**: Client logos and social proof elements\n\n**Features**:\n- Generated logos from company names\n- Verification badges\n- Industry categorization\n- Animated marquee\n- Trust indicators\n\n**Usage**:\n```tsx\nimport { LogoPlaceholder, TrustedBySection } from '@/components/ui/logo-placeholder';\n\n<LogoPlaceholder name=\"Company\" verified={true} />\n<TrustedBySection />\n```\n\n**Special Components**:\n- `ClientLogoGrid`: Grid of client logos\n- `TrustedBySection`: Complete \"trusted by\" section\n- `LogoMarquee`: Animated Staggered entrance animations\n- `slide-in-left/right`: Directional entrance\n- `glow`: Glowing border effect\n\n**Utilities**:\n- Glassmorphism effects (`.glass`, `.glass-strong`)\n- Gradient text utilities\n- Interactive button effects\n- Responsive animations\n- Accessibility considerations (reduced motion)\n\n## Design System\n\n### Color Palette\n\n```css\n/* Primary Gradients */\nfrom-blue-600 to-purple-600    /* Main brand gradient */\nfrom-purple-600 to-pink-600    /* Secondary gradient */\nfrom-pink-600 to-orange-600    /* Accent gradient */\nfrom-green-500 to-green-600 Typography\n\n- **Headings**: Bold, large scale with gradient text effects\n- **Body**: Clean, readable with proper line height\n- **UI Text**: Medium weight for labels and buttons\n\n### Spacing\n\n- **Cards**: Consistent padding (p-4, p-6, p-8)\n- **Gaps**: Systematic spacing (gap-4, gap-6, gap-8)\n- **Margins**: Generous whitespace for breathing room\n\n### Shadows and Effects\n\n- **Cards**: Layered shadows with blur effects\n- **Buttons**: Elevated shadows on hover\n- **Glassmorphism**: Backdrop blur with transparency\n\n## Accessibility Features\n\n### WCAG 2.1 AA Compliance\n\n- **Color Contrast**: All text meets minimum contrast ratios\n- **Focus Indicators**: Visible focus states for keyboard navigation\n- **ARIA Labels**: Proper labeling for screen readers\n- **Touch Targets**: Minimum 44px touch targets on mobile\n- **Motion**: Respects `prefers-reduced-motion`\n\n### Semantic HTML\n\n- Proper heading hierarchy\n- Landmark regions (main, section, nav)\n- Button effects\n\n### Animations\n\n- GPU-accelerated transforms\n- Efficient keyframe animations\n- Conditional loading based on `prefers-reduced-motion`\n\n### Responsive Design\n\n- Mobile-first
